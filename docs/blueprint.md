# **App Name**: Starlight Portfolio

## Core Features:

- Hero Section: Displays a welcoming introduction with name, title, and brief summary. Includes prominent buttons for resume download and project viewing.
- Career Summary: Summarizes the professional journey with key experiences and roles.
- Skills Showcase: Highlights key skills in AI, ML, React, and Cloud, presenting proficiency and marketability.
- Project Spotlighting: Focuses on featured projects, detailing contributions and outcomes with links to live demos or repositories.
- Research Highlighting: Showcases the research project on Facial Emotion Recognition Using AI, including methodology and results.
- AI-Powered Personalization Tool: Suggests relevant content based on an analysis of user behavior or interests using machine learning. Acts as a tool for user engagement.

## Style Guidelines:

- Background: Deep navy/midnight gradient for a professional and modern look. (#191970 to #000040)
- Primary: Subtle purple accents (#A020F0) to complement the dark background. (258, 79%, 53%)
- Accent: Cyan gradient highlights (#00FFFF) for interactive elements. (180, 100%, 50%)
- Body font: 'Inter', sans-serif, for clean and modern readability.
- Headline font: 'Space Grotesk', sans-serif, for bold, highlighted name and section titles.
- Minimal, elegant icons that align with the tech aesthetic (e.g., circular or outlined icons).
- Clean and balanced layout with ample spacing for a product-company level UI.
- Subtle starfield or tech-inspired background animation to add depth without being distracting.
- Smooth hover transitions on interactive elements to enhance user experience.
